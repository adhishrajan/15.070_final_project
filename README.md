# Cardinality-Constrained Portfolio Optimization

**Course:** Discrete Probability and Stochastic Processes — MFIN Spring 2026  
**Authors:** Aashrit Cunchala, Aymen Marzak, Adhish Rajan, Jean SainteBeuve, Clarys Vignal

---

## Overview

Empirical study of cardinality-constrained mean-variance portfolio optimization on S&P 500 daily returns (2015–2026, N=462 stocks, T=2854 days).

The project compares:
- **Brute force** exact optimum (small sub-universe, k≤5)
- **Greedy** forward selection (full N=462)
- **Simulated Annealing (SA)** (full N=462)
- **Closed-form Markowitz upper bound** (no cardinality constraint)

The paper (`main.tex`) is typeset in LaTeX and compiled on Overleaf.

## Running the Notebook

From the project directory:

```bash
python -m jupyter nbconvert --to notebook --execute --inplace \
    --ExecutePreprocessor.timeout=1800 portfolio_study.ipynb
```

Expected runtime: **10–15 minutes** on a modern laptop.

After execution, figures are saved to the root folder. Move them to `figures/`:

```powershell
Move-Item fig*.png figures\
```

Or on Linux/macOS:

```bash
mv fig*.png figures/
```

## Model

Cardinality-constrained mean-variance optimization (short selling allowed):

```
max  μᵀw
s.t. wᵀΣw ≤ σ₀²,   1ᵀw = 1,   |supp(w)| ≤ k
```

The inner problem (fixed k-subset) has a closed-form solution via Lagrangian duality. See §1.1 of the paper.
